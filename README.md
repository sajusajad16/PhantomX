# PhantomX Framework 💀

PhantomX is a modular cybersecurity framework for:
- Reconnaissance
- Port Scanning
- Web Scanning
- Vulnerability Detection
- Dashboard UI

---

## 🚀 Installation & Usage

### 1. Clone the Repository
```bash
git clone https://github.com/sajusajad16/PhantomX.git
cd PhantomX



2. Create Virtual Environment
python3 -m venv venv
source venv/bin/activate

3. Install Dependencies
pip install -r requirements.txt

4. Run PhantomX
🔹 CLI Mode
python main.py

🌐 Dashboard Mode
python dashboard.py

Open in browser:
http://127.0.0.1:5000

⚠️ Requirements
Python 3.x
Linux / Kali Linux recommended
